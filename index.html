2. DATA CLEANING & OPENREFINE

  • Primary Use: Cleaning messy data, de-duplication, standardizing inconsistent text (e.g., "Google", "google", "Google Inc.").

3. PANDAS (PYTHON DATA ANALYSIS)

Unique Values (The #1 Trap):
df['col'].unique(): Returns an array/list of the distinct values themselves.
df['col'].nunique(): Returns the integer count of unique values.
Trap: distinct() is SQL, not Pandas.
  • Frequency: df['col'].value_counts() gives counts of unique items.
  • Large Files: If a CSV is too big for memory, use chunksize in read_csv.
  • Code: pd.read_csv('file.csv', chunksize=10000).
  • Read Formats: read_csv, read_excel, read_json, read_parquet.

4. DATA VISUALIZATION & STORYTELLING

Chart Selection Rules:

  • Bar Chart: Comparing Categorical data (e.g., Sales by Region).
  • Line Chart: Trends over Time (e.g., Stock prices over a year).
  • Pie Chart: Proportions / Parts of a whole (e.g., Market share).

  • Histogram: Distribution of a single numerical variable.

Storytelling Principles:

  • Goal: Communicate insights, not just show data.
  • Narrative: Must have a beginning (context), middle (analysis), and end (actionable conclusion).

  • Human Element: Good stories highlight the impact on real people.

5. EXCEL FEATURES (EXAM TRIGGERS)

  • Forecast Sheet: Used for predicting future trends based on historical time-series data.
  • Text to Columns: Splits a single column into multiple based on a delimiter (comma, tab, space).
  • Trap: If you select "Delimited" but forget to check a specific box (e.g., Comma), the text remains unchanged.
  • Pivot Table: Summarizes large datasets (aggregation, grouping).
  • Trigger Word: "Summarize", "Aggregate".
  • VLOOKUP: Finds a value in a table row by matching a value in the first column.
  • WEEKNUM: Returns the week number of a date.
  • Trap: WEEKNUM(date, 2) changes the week start day.

6. WEB, APIs, & SCRAPING

Chrome DevTools (F12):

  • Network Tab: Inspects API calls, HTTP requests (GET/POST), response status, and payloads.
  • Application Tab: Inspects Cookies, Local Storage, Session Storage.
  • Console Tab: Shows JavaScript errors and logs.

HTTP Status Codes:

  • 200-299: Success (OK, Created).
  • 3xx: Redirection.
  • 4xx: Client Error (404 Not Found, 429 Too Many Requests).
  • 5xx: Server Error (500 Internal Server Error).

HTTP Methods:

  • GET: Retrieve data (idempotent).
  • POST: Send data to create a resource.
  • PUT: Update a resource.
  • DELETE: Remove a resource.

Scraping Ethics:

  • Robots.txt: The file that tells scrapers what they can and cannot scrape. ALWAYS check this.
  • Rate Limiting: Do not overload the server (avoid DDoS behavior).
  • Privacy: Do NOT scrape personal data (PII) without consent.

Tools:

  • BeautifulSoup: Parsing HTML/XML (static sites).
  • Selenium/Playwright: Browser automation (dynamic/JS-heavy sites).
  • Scrapy: Large-scale scraping framework.

8. MACHINE LEARNING CONCEPTS

K-Means Clustering:

  • Problem: Results can vary based on random initialization of centroids.
  • Solution: Set the n_init parameter (runs algorithm multiple times and picks the best).
  • K: Number of clusters. Small K = Underfitting risk? No, small K is usually distinct. Large K → Overfitting (clusters become single points).

ARIMA (Time Series):

  • Notation: ARIMA(p, d, q) or order=(p, d, q).
  • p: Autoregression (lags).
  • d: Differencing (integration/stationarity).
  • q: Moving Average.
  • Trap: Options often scramble this order (e.g., (d, p, q) is WRONG).

Statistics:

  • p-value < 0.05: Statistically Significant. (Reject Null Hypothesis).
  • Correlation (r):
    • +1: Perfect Positive.
    • -1: Perfect Negative.
    • 0: No linear relationship.
  • Note: Correlation does NOT imply causation.

9. GEOSPATIAL ANALYSIS

Libraries:

  • GeoPandas: Spatials operations (extends Pandas). Handles Shapefiles (.shp).
  • Folium: Interactive maps (Leaflet.js wrapper).
  • Shapely: Geometric objects (Point, Polygon).

Haversine Formula:

  • Purpose: Calculate the Great Circle Distance between two points on a sphere (Earth) given Latitude and Longitude.
  • Why not Euclidean? Euclidean distance creates a straight line through the earth (tunnel), which is wrong for surface travel.

Nominatim: API for geocoding (Address → Lat/Lon) and reverse geocoding.

10. LLMs & PROMPT ENGINEERING

  • Role of System Prompt: Defines the AI's persona, constraints, and behavior.

Educational AI (EduBot) Rules:

  • Constraint: Do NOT give final answers or full code.
  • Method: Guide the student, ask reflective questions, encourage critical thinking.

RAG (Retrieval-Augmented Generation):

  • Workflow: User Query → Vectorize Query → Search Vector DB → Retrieve Context → Send Context + Query to LLM → LLM Response.
  • Benefit: Reduces hallucinations, uses specific/private knowledge.

Prompting Strategy:

  • Specific > Generic: "List 3 benefits of solar" is better than "Tell me about energy".
  • Context: Providing context improves output quality.
  • Cost: Longer prompts/responses = More tokens = Higher cost.

11. COMMAND LINE (BASH & AWK)

Bash Basics:

  • ls -la: List all files (including hidden).
  • grep "pattern" file: Search for text.
  • | (Pipe): Passes output of one command as input to the next.
  • mv: Move or Rename.

Awk: Powerful text processing.

  • NR: Number of Records (current line number).
  • $1: First column.
  • {sum+=$1} END {print sum}: Sums the first column.

12. LOG ANALYSIS (SCENARIO)

Parsing Logic (Python):

  • Split string by spaces or regex.
  • Timestamp: Need to handle timezone offset (%z).

Filtering:

  • Success: 200 <= status < 300.
  • Method: GET or POST.
  • Time: Extract hour from parsed datetime object.

13. "NEVER CONFUSE" PAIRS

  • unique() (List) vs nunique() (Count).
  • Correlation (Strength of relationship) vs p-value (Significance of result).
  • Network Tab (API traffic) vs Application Tab (Storage/Cookies).
  • Git Pull (Download) vs Git Push (Upload).
  • Euclidean Distance (Flat plane) vs Haversine Distance (Sphere/Earth).
